**Key Updates: Stable Data Storage, Task Editing & Subtask Management**

This release resolves persistence issues and significantly improves task management flexibility.

### âœ¨ New Features
1.  **Stable File Storage**:
    *   Replaced `localStorage` with `electron-store` for robust file-based persistence.
    *   **Impact**: All data is now securely saved to the local disk, ensuring no data loss upon app restart.
2.  **Task Editing & Subtask Management**:
    *   **Click to Edit**: Tasks in the list are now interactive. Click any task to open a detailed modal for quick edits.
    *   **Subtask Quick Access**: Added subtask progress badges (e.g., "1/3"). You can now expand and toggle subtasks directly from the list view.
3.  **Task Priority & Sorting**:
    *   Added priority levels (High, Medium, Low) with visual indicators.
    *   Automatic sorting: Uncompleted tasks first, ordered by deadline proximity, then by priority.
4.  **Project Details Persistence**:
    *   Fully integrated persistence for Project Kanban boards, ensuring tasks are saved per project context.

### ğŸ› Fixes & Improvements
*   **Fixed**: Resolved a clipping issue where the date picker was hidden or cut off in the task detail modal.
*   **Fixed**: Resolved a bug where tasks were invisible in "Timeline" and "Calendar" views due to date serialization.
*   **Improved**: Enhanced the `DateTimePicker` UI for better responsiveness and precise centering of time scrollers.
