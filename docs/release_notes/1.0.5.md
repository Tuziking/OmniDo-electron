**Main Updates: Interactive Kanban Subtasks & Dynamic Progress**

### âœ¨ New Features & Improvements
1.  **Interactive Kanban Subtasks**:
    *   **Dropdown Management**: Click the subtask badge (e.g., `2/3`) on a card to reveal a dropdown list, keeping your board clean.
    *   **Independent Action**: Toggle subtasks directly on the board without opening the edit modal.
2.  **Dynamic Project Progress**:
    *   Project card progress bars now reflect real-time task completion statistics instead of placeholder data.
3.  **Clean Slate Projects**:
    *   New projects start completely empty, removing the need to delete default example tasks.
4.  **Keyboard Support**:
    *   Press `Enter` while editing a subtask to instantly create a new one, speeding up your workflow.

### ğŸ› Fixes
1.  **Modal Trigger Fix**:
    *   Resolved an issue where clicking a subtask accidentally opened the parent task's edit modal. Subtask interactions are now completely isolated.
2.  **Performance**:
    *   Fixed console warnings related to list animations (React Ref Forwarding) for better stability.
